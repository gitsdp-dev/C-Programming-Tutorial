# What Is C??
C is a **foundational, general-purpose programming language**. It was developed in 1972 by Bell Labs. C is a structured, HLL (High Level Language) which is **machine-independent**. It is **robust and portable programming language**.

C programming allows us to access low-level memory structures and hardware, making it incredibly fast and efficient to write the base, or core of a system software. C is called the **Mother Of All Programming Languages** as syntax patterns oc C influenced major programming languages developement such as **Java**, **C++**, **Rust**, **C#** , **PHP**. Even a popular Programming language named as **Python** is written using C. 

ANSI C. C99 and C23 are the most stable and the standardised versions of C till now, in which, C23 is a latest stable build of C.
